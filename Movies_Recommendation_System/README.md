# Movies Recommendation System

This project demonstrates a basic movie recommender system using content-based filtering techniques.

## Features
- Movie similarity based on genres, keywords, and metadata
- Cosine similarity to generate recommendations
- Recommend similar movies to the one selected
- Uses Pandas, Scikit-learn, and NLP preprocessing

## Dataset
- Source: [TMDB 5000 Movies Dataset](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)
- Fields: Title, Overview, Genres, Cast, Crew, Keywords
